# Fact-Check Agent – Truth Layer

An AI-powered fact-checking platform designed to verify claims extracted from PDF documents against real-time web data. The application combines Large Language Models (Claude AI) with live search capabilities to identify factual inaccuracies, outdated information, misleading statistics, and unsupported claims.

Users can upload research papers, business reports, articles, press releases, financial documents, or academic content. The system automatically extracts checkable statements, searches trusted online sources for supporting evidence, and generates detailed verification reports with confidence scores and explanations.

### Key Features

* PDF claim extraction using AI
* Live web verification with Google Custom Search
* Evidence-based fact checking
* Automated claim classification
* Confidence scoring for each verdict
* Interactive verification reports
* Modern Streamlit web interface

### Verdict Categories

* ✅ **Verified** – Evidence confirms the claim.
* ⚠️ **Inaccurate** – Claim is partially correct or outdated.
* ❌ **False** – Evidence contradicts the claim.
* ❓ **Unverifiable** – Insufficient reliable evidence found.

### Applications

* Academic research validation
* Business intelligence verification
* News and media fact-checking
* Market research analysis
* Corporate report auditing
* Financial statement verification
